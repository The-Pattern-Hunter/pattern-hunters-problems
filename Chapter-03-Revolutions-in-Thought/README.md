## 🎯 Learning Objectives

- Calculate Hardy-Weinberg equilibrium frequencies
- Simulate population genetics across generations
- Test for deviations from genetic equilibrium
- Model natural selection effects on allele frequencies
- Analyze real conservation genetics data
- Apply population genetics to medical and conservation problems
